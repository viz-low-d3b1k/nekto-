# Nekto+ 
Тема для угарного базара в голосовом чате некто.

- Для [Firefox'a](https://addons.mozilla.org/en-US/firefox/addon/nekto-plus/).        
- Для Chrome: денег нет. Но можно установить как распакованный аддон в режиме разраба.
